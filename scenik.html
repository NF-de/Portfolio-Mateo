Introduction

Est une application en ligne permettant de créer des plans feux à l’aide de plusieurs outils disponibles, tels qu’un calendrier, un chat pour communiquer avec le client, un enregistrement externe en PNG/PDF, un export XML afin de ne pas surcharger la base de données, ainsi que la gestion des projecteurs (déplacement/rotation) pour créer et personnaliser le plan selon ses besoins.

01.

Objectifs du projet :

L’objectif de ce projet est de développer une application web en ligne avec des comptes utilisateurs, permettant de partager rapidement les plans avec les clients. Grâce à un système de messagerie intégré, les échanges sont facilités.

02.

Outils et technologies utilisés

Utilisation des langages Web HTML, CSS, JavaScript et PHP pour le développement.

03.

Page de connexion

Une page de connexion permet à l’utilisateur de s’identifier.

04.

Page client

La page client affiche les informations du spectacle associé.

05.

Page administrateur

La page administrateur donne accès à l’ensemble des fonctionnalités.

06.

Chat

Ajout du chat côté admin et client, avec affichage du nom, date et suppression automatique des messages tous les 30 jours.

07.

Page Backend

Gestion des utilisateurs, modification des rôles et suppression des comptes.

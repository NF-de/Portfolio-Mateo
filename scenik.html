00.

Introduction

Est une application en ligne permettant de créer des plans feux à l’aide de plusieurs outils disponibles, tels qu’un calendrier, un chat pour communiquer avec le client, un enregistrement externe en PNG/PDF, un export XML afin de ne pas surcharger la base de données, ainsi que la gestion des projecteurs (déplacement/rotation) pour créer et personnaliser le plan selon ses besoins.

01.

Objectifs du projet :

L’objectif de ce projet est de développer une application web en ligne avec des comptes utilisateurs, permettant de partager rapidement les plans avec les clients. Grâce à un système de messagerie intégré, les échanges sont facilités pour recueillir des retours pour modifications.

02.

Outils et technologies utilisés

Utilisation des langages Web HTML, CSS, JavaScript et PHP pour le développement de l’application.

03.

Page de connexion

Une page de connexion permet à l’utilisateur de s’identifier.

04.

Page client

La page client affiche uniquement les informations qui lui sont attribuées, notamment les plans liés à son spectacle.

05.

Page administrateur

La page administrateur donne accès à l’ensemble des fonctionnalités : visualisation, modification, exportation, etc.

06.

Chat

Ajout du chat côté admin et client, avec affichage du nom, date et gestion auto des messages.
